# IKEA_LED_STM32

PCB with STM32 and PCA9685 to drive LEDs from IKEA LEDLJUS (24 or 64).
![img](https://github.com/MariusLamm/IKEA_LED_1.0_STM32/blob/master/IKEA_LED_PCB.PNG)

- GNU Make 3.81
